Create a env file at root directory
Then add this 
 # Gemini API Key // generate it from google ai studio
GEMINI_API_KEY="Paste your api key here"
USE_GEMINI=true
Then check in the askai folder that what model of gemini is running or what models are available

Then in one terminal run the server with 
npm run server 

in second terminal run these commands
# Diet-planner-app
install modules Run command npm install
Then run code by npm start
